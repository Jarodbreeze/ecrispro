Questions fréquentes
FAQ
Ai-je besoin d'une clé API pour utiliser ÉcrisPro ? ▾
Dans la version actuelle (bêta), oui — tu dois entrer ta propre clé API Anthropic. Dans la version finale, tu n'en auras pas besoin : le coût API est inclus dans l'abonnement mensuel.
Le contenu généré est-il unique ? ▾
Oui. Chaque génération produit un contenu original adapté à ton sujet, ton secteur et ton ton. Tu peux aussi générer des variantes pour A/B tester tes contenus.
Puis-je annuler mon abonnement à tout moment ? ▾
Absolument. Pas d'engagement minimum, pas de frais de résiliation. Tu annules en un clic depuis ton espace client.
Mes données sont-elles en sécurité ? ▾
Oui. Tout reste sur ton navigateur — aucun serveur intermédiaire ne stocke ton contenu ou ta clé API. Nous ne collectons aucune donnée de navigation.
Puis-je l'utiliser pour plusieurs clients (agence) ? ▾
Oui, le plan Agence est conçu pour ça. Il inclut une version white-label (ton logo, tes couleurs) que tu peux proposer directement à tes clients.
